1. O que é JavaScript e como funciona no navegador?

JavaScript (JS) é uma linguagem de programação usada para tornar páginas web interativas. Ele é executado diretamente no navegador e pode ser usado para:

  • ✅ Manipular elementos da página (mudar texto, cor, imagens, etc.).
  • ✅ Criar interações como botões clicáveis e formulários.
  • ✅ Fazer requisições para servidores e exibir dados dinamicamente.

Como o JavaScript é executado no navegador?

Quando um site é carregado, o navegador lê o código HTML e CSS, depois executa o JavaScript. Podemos incluir o JavaScript de duas formas:

Dentro da tag <script> no HTML

                 <script>
                        alert("Olá, mundo!");
                    </script>

Esse código exibe um alerta na tela.

Em um arquivo externo (script.js)

                
                    <script src="script.js"></script>

Isso mantém o código organizado e reutilizável.

Exemplo no navegador

Se você abrir o console do navegador (pressionando F12 e indo para "Console"), pode executar JavaScript diretamente, por exemplo: 

                
                    console.log("Hello, World!");
                

            
Isso exibe "Hello, World!" no console.
